# ✈️ Airline Operations Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze airline operations data. It provides insights into flight performance, delays, cancellations, and overall operational efficiency using visual analytics.

## 🎯 Objectives
- Analyze airline operational performance.
- Identify delay and cancellation patterns.
- Monitor key performance indicators (KPIs).
- Support data-driven decision-making.

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query
- DAX

## 📂 Project Structure
- Dashboard/ – Power BI (.pbix) file
- Dataset/ – Excel dataset
- Images/ – Dashboard screenshot

## 📊 Key Features
- Interactive filters and slicers
- KPI cards
- Trend analysis
- Delay and cancellation analysis
- Airline performance comparison
